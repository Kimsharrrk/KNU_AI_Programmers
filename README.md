## 실행 방법

```bash
# 가상환경 실행 후
pip install -r requirements.txt

# 의존성 설치 이후
python app.py
```

## 실행 결과
```bash
====================================
🧑‍🍳 자취생 전용 셰프 에이전트의 답변
====================================
---
## 📊 현재 요리 조건

| 항목        | 내용
|------------|----
| 🧊 남은 재료 | 계란 2개, 대파 조금, 스팸 반 통
| ⏳ 제한 시간 | 15분
| 🍳 요리 실력 | 완전 쌩초보

---

## 🍳 추천 메뉴: 스팸계란밥
이 간단한 스팸계란밥은 완전 쌩초보분도 쉽게 만들 수 있으며, 15분 안에 후딱 해먹을 수 있습니다. 계란과 스팸의 풍미가 입안에서 터져요!

---

## 📝 조리 순서

|단계| 조리 과정
|---|-----------
| 1 | 밥을 준비합니다.
| 2 | 계란을 풀어주세요.
| 3 | 대파를 잘게 잘라주세요.
| 4 | 팬에 기름을 두르고 대파를 볶습니다.
| 5 | 스팸을 넣고 볶습니다.
| 6 | 계란을 넣고 스팸과 대파와 함께 볶습니다.
| 7 | 밥에 계란과 스팸, 대파를 넣어 混合합니다.

> **💡 셰프의 꿀팁**
> * 팬에 기름을 두르기 전에 약간의 물을 넣어 불이 붙지 않게 조심하세요.
> * 계란을 풀 때 살짝 흔들어 주면 더 부드럽게 풀립니다.
---
```
